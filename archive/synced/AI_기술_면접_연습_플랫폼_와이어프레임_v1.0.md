# AI 기술 면접 연습 플랫폼 와이어프레임

이 문서는 `docs/frontend/screen-spec.md`를 보조하는 저충실도 와이어프레임이다.
시각 디자인보다 화면 구조, CTA 위치, 상태 안내, 선행 조건 노출 방식을 맞추는 데 목적이 있다.

## 1. 공통 레이아웃

### 1.1 로그인 전
- 상단: 로고
- 중앙: 핵심 설명 + CTA
- 하단: 약관/개인정보 링크

### 1.2 로그인 후
- 상단: 글로벌 내비게이션
  - 포트폴리오
  - 지원 준비
  - 면접 히스토리
  - 내 정보
- 본문: 페이지 컨텐츠
- 우측 또는 하단: 상태 알림/실패 재시도 영역

## 3. 와이어프레임 사용 규칙

- 파일 제한, 질문 수 제한, 답변 길이 제한은 항상 입력 UI 근처에 보여준다.
- `건너뛰기`는 일반 제출 버튼과 분리한다.
- 세션 자동 일시정지와 복원 가능 시간은 세션 화면에 항상 노출한다.
- private repository는 배지와 안내 문구로 public과 구분한다.
- 추출 실패 문서는 단순 실패 표시로 끝내지 말고 품질 영향 메시지를 함께 보여준다.
